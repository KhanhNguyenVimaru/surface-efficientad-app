EfficientAD Web App - System Pipeline

Sơ đồ pipeline tổng quan và luồng xử lý chi tiết của endpoint /predict.

1) Luồng tổng quan

flowchart TD A[User mở Nuxt Frontend] --> B[Upload ảnh + chọn threshold] B --> C[POST /predict multipart form] C --> D[FastAPI validate input] D --> E[Load ảnh PIL RGB] E --> F[Run EfficientAD inference từ checkpoints/capsule.ckpt] F --> G[Trích xuất pred_score / pred_label / anomaly_map / pred_mask] G --> H[Phân loại defect type] H --> I[Tạo ảnh trực quan hóa\n anomaly map / overlay / mask overlay] I --> J[Classical CV hậu xử lý\n đếm lỗi, contour, largest area] J --> K[Trả JSON response chứa metadata + ảnh base64] K --> L[Frontend hiển thị kết quả]

2) Chi tiết backend /predict

flowchart TD A[Nhận UploadFile image, threshold] --> B{anomalib import OK?} B -- No --> B1[HTTP 500] B -- Yes --> C{checkpoint capsule.ckpt tồn tại?} C -- No --> C1[HTTP 400] C -- Yes --> D{threshold >= 0?} D -- No --> D1[HTTP 400] D -- Yes --> E[Đọc bytes + parse ảnh] E --> F{Ảnh hợp lệ?} F -- No --> F1[HTTP 400] F -- Yes --> G[Engine.predict] G --> H{Inference thành công?} H -- No --> H1[HTTP 500] H -- Yes --> I[Chuẩn hóa output model] I --> J[Classify GOOD/DEFECT + defect_type] J --> K[Generate base64 visual outputs] K --> L[Classical CV postprocess] L --> M[Build PredictResponse]
